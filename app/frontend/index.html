Lab context

Các thông tin này giúp bạn demo và giải thích kết quả ở các tuần sau.

Frontend routing Frontend gọi API qua /api/* và /debug* cùng origin.
Backend proxy Nginx chuyển tiếp request tới api-service.vulnerable.svc.cluster.local:8080.
Baseline weakness API đang chạy root, token service account được mount, và có endpoint debug phục vụ lab.
Useful demo flow Health → Profile → Debug → Exec → sau đó thêm RBAC, NetworkPolicy, Pod Security, detection.
Gợi ý mở rộng cho các tuần sau: thêm secure/vulnerable toggle, badge version image, trạng thái policy, và bảng kết quả attack-defense theo từng domain.
